    • 人材育成・発掘

    リアルプロジェクト型の研修、才能発揮・発掘のための1on1、基礎スキル研修等の人材育成・発掘のプログラムを提供しています。 研修や1on1においては、徹底的に個人やチームと向き合うことで、組織としてのレベルアップを図ります。

    以下は人材育成プログラムの一例です。この他にも「こんな育成プログラムはできないか？」などのご要望がありましたらご相談ください。

    人材育成プログラム例

    • リアルプロジェクト型BizDev研修
    • 才能発揮・発掘のための1on1
    • 基礎スキル研修
    • etc…

    　

    Program 1

    リアルプロジェクト型BizDev研修

    BizDevリアルプロジェクト型研修では、机上に留まらないリアルな経験を通じて人材育成を行います。研修参加企業・組織のみならず、地域企業・地方自治体にとってもプラスになる形を目指します。

    研修としてではなく実プロジェクトとして、短期間で徹底的にリアルな課題解決に向き合うプログラムです。戦略・企画力、観察力、現場に入り込む力、チーム形成力、実行力、行動力、計画力の全てが試されるハードなものですが、このプログラムを経ることで、今後のプロジェクト推進の素地が出来上がります。企画から当日のディレクション、振り返りまでを一緒に走り切ります。

    また、地方との協業をお考えの企業様につきましては、本プログラムを通じて双方の理解を深める機会ともなりますので、ぜひご検討ください。

    　

    Program 2

    才能発揮・発掘のための1on1

    個人や組織の状況に応じた1on1を行います。
    対話をしながら、その時のご本人に合うテーマやスタイル（コーチング、ティーチング、壁打ちなど）での1on1を提供します。

    （1on1の対象者例）

    • マネジメント経験が長い方
    • 部下やチームを持ち始めた方
    • 中途入社、所属変更などで大きく環境が変わった方
    • 普段実務に追われ、業務から一歩離れて自身の思考を整理したり、振り返ったりするタイミングを掴めていない方
    • 思考がまとまっていない状態で人に相談・壁打ちすることに抵抗がある方
    • etc…

    　

    Program 3

    基礎スキル研修

    何かを「できる」ようになるためには、知識獲得だけでなく「実感を持って分かる」経験が大切です。 実感を持って分かるためには、失敗しても良い場で小さくトライし、必要なスキルと自身の現在地のギャップを把握する必要があります。

    基礎スキル研修では、トライ→失敗→ギャップ認識の経験を通じて「実感を持って分かる」環境を提供し、研修参加者が今後の実務の中で自律的にスキルアップするきかっけをつくります。

    （基礎スキル研修の例）

    • 論理的思考
    • 仮説設計・検証
    • リサーチ設計
    • 数値計画策定
    • ドキュメント作成
    • etc…
